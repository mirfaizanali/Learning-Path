The Roadmap

3-Month Sprint Focus weeks 1–12 on Phases 1 & 2. At ~5 hrs/week, that's realistic. Phases 3–5 continue the journey over months 4–7.
01

Spring Boot Mastery

Weeks 1–5 · 25 hrs ★ Sprint Phase Build on Spring Core
Auto-configuration REST APIs Spring Data JPA Hibernate Validation Spring Security + JWT Lombok Actuator Swagger / OpenAPI
Secure Task Manager REST API with JWT auth, MySQL, and Swagger docs — tested with Postman, deployed locally.
02

Angular Fundamentals

Weeks 6–12 · 35 hrs ★ Sprint Phase Frontend track
TypeScript Angular CLI Components & Directives Services + DI Routing Reactive Forms HttpClient RxJS basics Full-stack integration
Angular frontend wired to your Phase 1 API — deployed to Netlify. First true full-stack project shipped.
03

Microservices & DevOps

Weeks 13–18 · 30 hrs Spring Pro prep Continuation track
Microservices principles Spring Cloud API Gateway Eureka Docker Docker Compose GitHub Actions CI/CD JUnit 5 + Mockito
Split monolith into 2 microservices, containerize with Docker, set up CI pipeline in GitHub Actions.
04

AI for Java Devs DIFFERENTIATOR

Weeks 19–23 · 25 hrs ★ Portfolio Highlight High-signal skill
LLM fundamentals Prompt engineering Spring AI LangChain4j OpenAI / Anthropic APIs RAG pattern Vector DBs (pgvector) Embeddings Function calling
Add an AI assistant to Task Manager — RAG over user tasks. This is what sets your resume apart in 2026.
05

Cloud + Capstone + Interviews

Weeks 24–28 · 25 hrs AWS CCP cert Job-ready
AWS EC2 S3 RDS IAM Deployment System design basics DSA refresh Mock interviews
Deploy full-stack app to AWS. Polish GitHub. Pass AWS Cloud Practitioner. Update resume. Apply.
